# Ho Chi Minh Vegan Restaurant Analysis

**Description:** This project involves cleaning and analyzing listings data which was scraped directly from Google Maps platform. The dataset includes 18 columns and 141 restaurant listings, but it contains several formatting inconsistencies, missing values, duplicated information, and unstandardized categories that need to be cleaned before analysis. Finally, a scoring model is provided to evaluate key decision factors including listing density, review volume, price segment, competition intensity, customer fit, and concept fit.

The objective is to generate business insights to help a client choose the best location and restaurant model for a vegan concept in Ho Chi Minh City targeting foreign customers. The analysis focuses on market demand, competition, pricing, restaurant styles, customer preferences, and location attractiveness to support data-driven recommendations.

**Tools:** Excel, Power Query and PowerPoint.

**Skills:** Web data scrapping, Data cleaning (e.g.: nulls & duplicates handling, data correction, standardize data type), Append & merge tables, XLOOKUP referencing, categorizing, descriptive statistics, PivotTables & PivotCharts, conditional formating, process documenting, charts building, scorecard model building, recommendation and presentation.

**Outputs:**  
- Excel workbook in `.xlsx` format with multiple tabs covering raw data, cleaned data, data dictionary, data handling documentation, pivot tables, charts and scorecard.
- Presentation slides in `.pdf` format containing problem statement, objectives, data overview, insights, scorecard, recommendations, and additional remarks.

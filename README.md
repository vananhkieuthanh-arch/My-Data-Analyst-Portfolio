## Projects Summary

Short summary of the listed projects is provided for quick comprehension. For more details, please check out the documents in each project folder.

#### SQL + Power BI | Olist Brazilian E-Commerce Analysis

Analyzed approximately 100,000 Olist e-commerce orders from 2016–2018 to understand sales performance, customer satisfaction, and delivery operations. This project demonstrates SQL data preparation in Google BigQuery, Power Query transformation, Power BI data modeling, DAX measures, KPI tracking, Dynamic filtering, YoY analysis, and dashboard storytelling for business decision-making.

#### SQL + Looker Studio | Travel Booking App Analysis

Analyzed 19,096 travel booking records to monitor business performance, customer behavior, route patterns, and growth opportunities. This project demonstrates exploratory data analysis, calculated fields, segmentation, dashboard building in Looker Studio, and translating confidential business data into clear insights and recommendations.

#### Excel + Power Query | Ho Chi Minh Vegan Restaurant Analysis

Cleaned and analyzed Google Maps listing data for 141 vegan/vegetarian restaurants in Ho Chi Minh City to support a client’s market-entry decision. This project demonstrates web-scraped data cleaning, Power Query, lookup functions, PivotTables, PivotCharts, descriptive analysis, scorecard modeling, and business recommendation development for location and restaurant concept selection.

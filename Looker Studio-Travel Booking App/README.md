# Travel Booking App Analysis

**Description:** This dashboard provides a comprehensive overview of travel booking performance to monitor business health, analyzing trends, customer behavior and identify growth opportunities.

**Tools:** Google Bigquery, Looker Studio.

**Skills:** EDA, calculated fields, segmentation, dashboard, story, and presentation.

**Outputs:** 
- Presentation slides in `.pdf` format (exported from Looker Studio) containing brief context, objectives, dataset details, insights and recommendation.

## Datasets
- Data source using is from confidential company data so raw dataset or exact dashboard cannot be shared 
- Dataset including passenger info, ticket info and price, depart from, arrival to destination
  - Fields: 20 columns
  - Rows: 19,096 rows
